
# “宏观数据 → 资产配置策略”可落地映射表

1. 宏观状态层（LLM 输出）
2. 信号 → 策略映射（核心表，规则+LLM）
3. 策略 → 资产权重调整（执行层）


# LLM 不直接“算权重”，而是“判断状态 + 调用策略模板”

宏观数据 → 指标加工 → 宏观状态识别（LLM）
        ↓
  策略池筛选（规则+LLM）
        ↓
  权重区间 / 调仓建议（量化引擎）
        ↓
  合规 & 解释层（LLM）



Decision （LLM/量化引擎） -> Execution （量化引擎） -> Explanation + Client Narrative(LLM)
