## 一、战略锚定层（Strategic Anchor）

**定位：长期收益来源，决定组合 90%+ 收益特征；低频调整（年 / 季）**

| 类别    | 策略名称             | 核心逻辑                | 关键注意事项                                      |
| ----- | ---------------- | ------------------- | ------------------------------------------- |
| 战略锚定层 | 60/40 股债（区间约束）   | 以股债分散获取长期稳健收益       | 🇨🇳 股债相关性时变、政策市特征；建议区间化（如股40–60%）；📦 理财/养老 |
| 战略锚定层 | 均值-方差优化（MVO）     | 用历史 μ/Σ 在约束下求最优权重   | ⚠️ 参数敏感、结构断点多；建议收缩估计/稳健优化；📦 私行/专户          |
| 战略锚定层 | Black-Litterman  | 市场均衡收益 + 主观观点生成后验收益 | ⚠️ 观点需量化与可验证；适合投委会框架；📦 私行/专户               |
| 战略锚定层 | 风险预算框架（SAA）      | 先定各资产风险贡献，再反推权重     | 🇨🇳 相关性突变风险高；需定期校准；📦 养老/稳健理财              |
| 战略锚定层 | 风险平价（等风险特例）      | 风险预算的等权版本，平衡风险来源    | ⚠️ 无衍生品易“伪RP”；需国债期货；📦 大资金                  |
| 战略锚定层 | 全天候（All Weather） | 覆盖增长/通胀升降四环境        | 🇨🇳 杠杆与期货替代是关键；注意基差与展期；📦 养老/稳健            |
| 战略锚定层 | 因子配置（长期倾斜）       | 长期偏向价值/质量/低波等因子     | ⚠️ A股因子时变与拥挤；需年度复检；📦 公募/量化                 |
| 战略锚定层 | 目标日期基金（TDF）      | 随年龄/期限递减权益仓位        | 🇨🇳 退休年龄/房产占比需本土校准；📦 个人养老金                |
| 战略锚定层 | 核心-卫星（核心仓）       | 核心仓提供稳定β            | 说明：卫星仓调用下层策略；📦 私行/专户                       |

| ----- | ---------------- | ------------------------ | ---------------------------------- |
| 战术增强层 | 宏观周期轮动           | 增长/通胀/流动性/信用 → 状态识别 → 轮动 | 🇨🇳 加入社融、M1-M2、政策修正；⚠️ 数据滞后；📦 理财 |

| 战术增强层 | 信用周期轮动           | 利差/违约变化切换债类资产            | 🇨🇳 城投区域分化；⚠️ 流动性分层；📦 固收+        |
| 战术增强层 | 动量与趋势策略          | 6–12M 时间序列动量/趋势          | ⚠️ 政策突变期失效；需双窗口+波动缩放；📦 CTA小容量     |
| 战术增强层 | 估值分位数调整          | 估值极端 → 均值回归/再定价          | ⚠️ 估值中枢迁移；需盈利过滤；📦 理财              |
